O que é design responsivo?

O design responsivo é uma abordagem de design que visa adequar páginas da internet ou sites a variados tipos de dispositivos e telas, oferecendo a melhor experiência de uso para o usuário.

CSS e design responsivo

Usa-se a união entre Layout fluido, imagens flexiveis e medias queries.

  • Layout fluido: Define-se o tamanho com % em vez de px, assim, preservam um alinhamento entre a relação do elemento pai e o elemento filho.
  • Imagens flexiveis: Imagens que não ultrapassam o tamanho do contêiner(pai). Usa-se a "max-width:100%", assim, a imagem nunca passará do pai.
  • Media queries: Usado para "adaptar" um arquivo html de acordo com o tamanho/resolução do dispositivo. Usa-se "@media (min-width: valor da tela do dispositivo). Também se altera, no caso de colunas, a 'width' da 'div'.

Flex-Box, o que é e como usar.

O flex-box é uma propriedade que usa os comandos'flex' do CSS. É dividido em Flex Container (usado quando tem 'display;flex') e Flex Item(São os filhos do flex container). Os flex container são:

  • Display: flex
  • Flex-direction
  • Flex-wrap
  • Justify-content
  • Align-items/li>
  • Align-content
